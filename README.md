# Smart Parking Space Detection System 

## Project Overview
The Smart Parking Space Detection System is designed to identify available and occupied parking spaces.
It helps reduce traffic congestion and saves time for drivers.

## Technologies (Planned)
- IoT Sensors / Camera
- Python / Java
- OpenCV (optional)
- Arduino / ESP module

## Use Cases
- Shopping malls
- Office parking
- Smart city applications
